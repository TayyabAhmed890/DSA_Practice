like:
array = [1, 3, 4, 6, 8, 10]

Approach No 1 (Sorted Array)
use two pointers approach

1) create two variables left = 0 and right = arr.length-1

2) start a while loop left < right
3) Find the sum of 1st and last index of the array
4) if sum = target return indexes 
5) else if sum < target left++
6) else if sum > target right--

7) return false
